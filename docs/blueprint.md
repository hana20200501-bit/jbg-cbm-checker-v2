# **App Name**: CBM Vision

## Core Features:

- Role Management: Role-based access: Manager and Worker roles with tailored interfaces.
- Dashboard: Dashboard view with shipper data, progress tracking, and total CBM calculation.
- Real-time Data Sync: Real-time updates of CBM data across devices using cloud sync.
- Offline Data Entry: Offline data caching: Allows workers to input data even without internet.
- CBM Calculation: CBM Calculator: calculate total CBM.
- Mobile Responsiveness: Mobile-friendly design with responsive UI components.
- Predictive Input: Smart Input Suggestion: A tool which predicts and suggests likely box dimensions based on historical data and shipper characteristics, reducing input time and errors. 

## Style Guidelines:

- Primary color: Moderate blue (#558bff) for a clean and efficient feel, leaning towards the classic CBM color but not too trendy.
- Background color: Light gray (#f2f5ff), a very desaturated hue of the primary color.
- Accent color: Violet (#a155ff), a lighter and more saturated analogous hue.
- Body and headline font: 'PT Sans' (sans-serif), offering a balance of modernity and approachability suitable for both headlines and body text.
- Simple, clear icons from the provided set, consistently styled across the app.
- Clean, card-based layout for clear information hierarchy and easy navigation.
- Subtle transition animations for modal appearances and data updates.
